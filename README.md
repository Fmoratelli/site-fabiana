# ✦ fab[IA]na · Creative Systems

> *"Recomeçar NÃO é começar do zero. É começar com EXPERIÊNCIA."*

Este repositório contém o código-fonte do site oficial de **Fabiana Moratelli** (`fabianamoratelli.com.br`) — um ecossistema que une **Alma Antiga e Ferramenta Nova**.

---

## 🧪 A Filosofia

- **Entregar ao robô o que é do robô:** Automações sob medida, agentes de IA defensivos e eliminação do trabalho operacional repetitivo.
- **Devolver ao humano o tempo de pensar:** Resgate do tempo livre para estratégia, criação, direção de arte e visão de futuro (*foresight*).
- **Âncora Humana:** 12 anos de repertório em fotografia, vídeo, branding e design garantindo que a tecnologia sirva à intenção, e não ao ruído genérico.

---

## 🛠️ Stack & Arquitetura

| Camada | Tecnologia / Ferramenta |
| :--- | :--- |
| **Frontend** | HTML5 Semântico, CSS3 Custom Variables, Vanilla JS (Canvas 2D Engine) |
| **Hospedagem & CDN** | Netlify Continuous Deployment (via GitHub Main Branch) |
| **Domínio & DNS** | Hostinger / Netlify Edge |
| **Infraestrutura de IA** | n8n (VPS Própria), APIs do Gemini/OpenAI, Evolution API |
